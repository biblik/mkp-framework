boolean edu.emn.mkp.Solution.validate ( ) throws Exception

Check that the solution is feasible for MKP. Tests are the following :

  • compute the value of the knapsack (evaluate);
  • check that each constraint is not violated.

Error messages are available with the method getError().

Returns
Return true if solution if feasible, false otherwise.

long [] edu.emn.mkp.Solution.m_weights
protected

Sum of the weights of the items in the knapsack following each dimension. For a solution, the computation of each weight is performed in validate().

For computational requirement, update this attribute iteratively with method setWeight(int, long). For further examples, see addItem(int) or removeItem(int)) .
